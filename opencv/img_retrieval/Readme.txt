Image Retrieval

Software Required:

Visual Studio 2019
Opencv 2.4.13.6-vc14
cvui 2.7.0

Environment Set Up:

1.Install opencv 2.4.13.6-vc14 at C: disk.

2.Add a new variable in user variables in "Environment Variable".
Variable: Path
Value: C:\opencv\build\x64\vc14\bin

3.Copy all the dll. file at C:\opencv\build\x64\vc14\bin to C:\window\system32 if you install and set up opencv 2.4.13.6 for first time.

4.Place the dataset file at C: disk.

5.Place the cvui-2.7.0 at C: disk.


Execution:

1.Open the Image_Retrieval programs using visual studio. 

2.Turn to Debug x64 mode.

3.Execute the program.

4.Select the target object to be retrived from the database in ui.

5.Press the Start Retrieval button to start the image retrieval in ui.

Output:

The output text will be display on the screen.
The retrieved images will be displayed on the screen.
The retrieved images can be found at C:\dataset\searchResults.
